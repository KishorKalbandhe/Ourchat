⚙️ One-time Setup

This app needs a free Firebase project to store your messages. It only takes 5 minutes!

1
Go to console.firebase.google.com and create a free project
2
Add a Web App to your project and copy the firebaseConfig values
3
Enable Firestore Database and Authentication → Email/Password
4
Paste your config values below

Your config is saved only on this device.
Share this HTML file + your config with her so she can do the same.
